# Task Name Generation Prompt

Generate realistic task names for a {department} team working on a {project_type} project at a B2B SaaS company.

## Requirements:
- Make task names specific and actionable
- Use professional language
- Include context relevant to the department
- Vary the complexity and scope

## Examples by Department:

### Engineering:
- Implement user authentication API endpoint
- Fix pagination bug in dashboard component
- Refactor notification service for better performance
- Add unit tests for payment module

### Marketing:
- Create blog post about Q4 product updates
- Design email campaign for product launch
- Analyze social media engagement metrics
- Plan webinar on industry trends

### Sales:
- Follow up with enterprise prospect Acme Corp
- Prepare demo for potential client meeting
- Update quarterly sales forecast
- Review and finalize contract terms

### Product:
- Write PRD for new analytics feature
- Conduct user interviews for feedback
- Prioritize backlog for next sprint
- Update product roadmap for stakeholders

Return only the task names, one per line.
